Anatomia de tag

  • Abertura e fechamento
  • Conteúdo, o conteúdo é o que está dentro da tag
  • Elementos, é a combinação de tag + conteúdo
  • Elementos vazios, Ex:
    • Tag img e input
    • Elementos vazios são aqueles que não possuem conteúdo dentro da tag e nem fechamento
    • Por essa ausência a tag faz uso de atributos
  • Exemplo de atributo: tag 'img src="" alt="" '
    • o ALT serve para adicionar um texto alternativo caso a imagem não aparece
  • Abributis booleanos: Verdadeiro ou falso. Ou a tag tem um certo atributo ou não
    • por exemplo o atributo DISABLE da tag INPUT

Atributos Globais

São atributos que podem ser aplicados a qualquer tipo de TAG

  • class="Atribuir uma classe para poder identifica-lo no CSS ou JS"
  • contenteditable
  • data-
  • hidden. Esconde a tag
  • id="Mesma coisa que class"
  • style="Tu já sabe"
  • tabindex="Quando vc aperta TAB ele vai selecionar na ordem de seleção de tag que você definiu"
  • title="definir um título á tag"
  • Procurar por mais atributos globais
