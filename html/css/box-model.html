Box Model é fundamental para fazer layouts para web e ele conssite em caixa 2D encima de caixas 2D.

As propriedades da caixa são: largura(width), altura(height), conteúdo(content), bordas(border), preenchimento(padding) e espaços externos(margin)

CSS é incrível

Quando você usa "box-sizing: border-box" em um elemento HTML, o tamanho total do elemento incluirá a largura e altura do conteúdo, a borda e o preenchimento. Isso ajuda a garantir que as dimensões do elemento permaneçam consistentes, mesmo quando você adiciona uma borda ou preenchimento. Em outras palavras, é uma maneira de controlar o tamanho do elemento de forma mais precisa.

O display block e inline vão declarar como as caixas vão se commportar em relação as outras(Comportamento Externo).
block Vai ocupar toda a linha, empurrando sempre o próximo elemento abaixo desse, width & heigth são respeitados e padding, margin e border vão funcionar normalmente, alguns elementos HTML por padrão já são block, mas é posssível alterar isso com "display: inline" por exemplo: <p>, <div>, <section> e <h1>

Já o display inline vai ocupar somente o espaço total do elemento fazendo com que seja possível posicionar elementos um ao lado do outro, width e heigth não funcoinam e somente os valores horizontais de margin, padding e border vão funcionar, ou seja, quando você declarar um "margin: 100px" o elemento vai dar um espaçamento externo somente da esquerda para a direita e não vai afetar a largura e altura. Exemplos de tags por padrão inline: <a>, <strong>, <span> e <em>.
