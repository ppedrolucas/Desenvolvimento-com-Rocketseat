Glosário de conceitos relevantes e principais tags do HTML

<em/>
Dar ênfase à uma parte específica do texto o deixando em Itálico. É importante deixar um texto em Itálico para destacar palavras-chaves ao leitor, ajudando-o a identificar informações importantes.
Semântica
É importante manter uma boa semântica no código HTML, para que seja fácil de ler e entender tanto para o desenvolvedor do projeto quanto para quem quiser analisá-lo.
<code/> e <pre/>
Ambos os códigos servem para representar um código na tela do navegar porém o primeiro só representa uma linha de código simples e genérica enquanto o outro é capaz de representar um bloco de código robusto e mais complexo acrescentando os recuos e identações do código no VS Code.
